# 🧮 Simple Calculator

A modern and responsive **Calculator Web Application** built using **HTML, CSS, and JavaScript**. This project performs basic arithmetic operations with a clean user interface and interactive user experience.

---

## ✨ Features

✅ Addition, Subtraction, Multiplication & Division  
✅ Responsive Design 📱💻  
✅ User-Friendly Interface 🎨  
✅ Real-Time Calculations ⚡  
✅ Clean and Organized Code 🧹  
✅ Lightweight and Fast 🚀  

---

## 🛠️ Technologies Used

- 🌐 HTML5
- 🎨 CSS3
- ⚡ JavaScript (ES6)
